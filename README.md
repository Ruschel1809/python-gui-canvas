# Python GUI & Canvas

Übungen und Transferaufgaben zu grafischen Benutzeroberflächen, Canvas
und Systemumgebung aus dem Python-Modul meiner Weiterbildung zur
**KI-Entwicklung** (Juni – Dezember 2025).

## Inhalte

| Ordner | Beschreibung |
|---|---|
| `PythonUebung10301Systemumgebung` | OS-Modul – Verzeichnisse erstellen, Dateien lesen und schreiben mit rekursiver Verzeichnisliste |
| `PythonUebung10302Systemumgebung` | OS-Modul erweitert – Dateien filtern, umbenennen und JSON-Operationen |
| `PythonUebung10303Systemumgebung` | OS und JSON kombiniert – persistente Datenspeicherung mit Aktualisierungsfunktion |
| `PythonUebung11601GUI` | Tkinter Einstieg – erstes GUI-Fenster mit Label, Entry und Buttons |
| `PythonUebung11602GUI` | Tkinter – Label und Button Interaktion |
| `PythonUebung12301Canvas` | Canvas – Zeichnen mit Oval und Linien, Radiobuttons zur Farbsteuerung |
| `PythonUebung12302GUI` | GUI mit Grid-Layout – Farbauswahl synchronisiert Canvas und Label |
| `Transferaufgabe111101` | Begrüßungs-App mit Entry, Radiobuttons, Zeitstempel-Log und Text-Widget |

## Highlights
- Robuste Fehlerbehandlung bei Datei- und JSON-Operationen
- `os.walk` für rekursive Verzeichnisnavigation
- Synchronisierung von Canvas und Label über Radiobutton-Auswahl
- Persistente JSON-Datenspeicherung mit Aktualisierungsfunktion

## Technologien
- **Sprache:** Python 3
- **Bibliotheken:** tkinter, os, json, datetime
- **IDE:** PyCharm
